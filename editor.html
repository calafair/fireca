Here's how to use the Fireca Editor:


Highlight a piece of text to see some formatting options (Bold text, italics, links, block quotes, etc.).

Use the shortcuts (Mac) command b to bold, command u to underline, and command i to italicize.

For other devices, use control b to bold, control u to underline, and control i to italicize.

Note: control/command z can be used to undo and control/command shift z can be used to redo.
Start writing here!
