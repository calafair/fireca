Spark UI

Spark UI is a user friendly way to style your pages. We will highlight some of the most important parts of Spark design.
Spark is similar to Flat Design, with transitions, and hover effects. It also incorporates transparency and blurring.

Click Effects

Spark UI click effects are subtle changes in opacity when a user clicks an element.

.sp-click:active {opacity: 0.5; transition: 0.2s;}




Blurring, Transparency, and Hover Effects

For transparency, use rgba values (red, green, blue, alpha). For the alpha value, set it to 0.7 or 0.8 and decrease to 0.4 or 5 on hover. Change the rgb values and opacity for a lighter hover effect. To blur, use backdrop-filter and -webkit-backdrop-filter (for cross-browser support). Like click effects, hover effects are subtle changes in the element's background, either a brighter or darker color. Transparency allows an element's background to be 'see-through', and blurring makes the text more readable, and still lets the element retain the colors of the background.

.sp-hover-dark {backdrop-filter: blur(100px); -webkit-backdrop-filter: blur(100px); transition: 0.15s; color: white; background-color: rgba(70, 90, 101, 0.75);}

.sp-hover-light {backdrop-filter: blur(100px); -webkit-backdrop-filter: blur(100px); transition: 0.15s; color: black; background-color: rgba(210, 210, 210, 0.7);}

.sp-hover-dark:hover {background-color: rgba(140, 130, 144, 0.45);}

.sp-hover-light:hover {background-color: rgba(250, 250, 250, 0.65);}

Links and Buttons

Here's how to create a link or a button:

.sp-link {cursor: pointer; text-decoration: underline; color: #2980b9;}
.sp-link:link {color: #2980b9;}
.sp-link:visited {color: #16a085;}
.sp-link:focus {color: rgb(60, 120, 240);}
.sp-link:hover {color: rgb(84, 160, 255);}
.sp-link:active {color: rgb(60, 130, 240);}

.sp-btn {-webkit-touch-callout:none;-webkit-user-select:none;-khtml-user-select:none;-moz-user-select:none;-ms-user-select:none;user-select:none;border:none;display:inline-block;padding:8px 16px;vertical-align:middle;overflow:hidden;text-decoration:none;color:inherit;text-align:center;cursor:pointer;}
